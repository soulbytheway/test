## 3. Розробка діаграм варіантів використання

```mermaid
flowchart TD
    %% Актори
    A[🧑‍💻 Користувач системи]
    B[🧑‍🔬 Рецензент]
    C[🌐 Зовнішні джерела]
    D[⚙️ Система]

    %% Прецеденти користувача
    A --> U1((Додавання джерела))
    A --> U2((Імпорт джерела))
    A --> U3((Створення тематичних колекцій))
    A --> U4((Анотування джерел))
    A --> U5((Форматування бібліографії))
    A --> U6((Створення цитат))

    %% Прецеденти рецензента
    B --> R1((Перевірка бібліографії))
    B --> R2((Експертиза джерел))

    %% Прецеденти зовнішніх джерел
    C --> E1((Надання метаданих))
    C --> U2

    %% Прецеденти системи
    D --> S1((Автоматичне оновлення метаданих))
    D --> S2((Синхронізація бібліотеки))
    D --> S3((Резервне копіювання))

    %% Внутрішні зв'язки
    U2 --> S1
    U1 --> S2
    U3 --> S3
